百词斩产品分析报告——交互式网页
本项目将《百词斩产品分析报告》以网页形式呈现，采用类苹果设计风格，提供分页滑动、导航跳转、Figma 原型嵌入等交互体验。

本地运行
1. 将本项目所有文件下载到本地
2. 确保 `index.html` 与 `bczProdctRsrch` 文件夹处于同一目录
3. 双击 `index.html` 即可在浏览器中打开
